Заготовка многопоточной программы, использующей библиотеку POSIX Threads, с примерами основных операций

addheaders - добавить заголовочные файлы для компиляции в системах без библиотек POSIX
remheaders - удалить заголовочные файлы (удалены по умолчанию)

используйте Misc\pthreadVCE2.lib и Misc\pthreadVCE2.dll для систем без библиотек POSIX
